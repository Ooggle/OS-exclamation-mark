# OS!
OS! is a kernel and OS development project

### Compilation dependencies	
```sh	
sudo apt install make nasm gcc genisoimage	
```	

### Emulation dependencies	
```sh	
sudo apt install bochs bochs-sdl	
```	

### Building and running	
```sh	
make run	
```
